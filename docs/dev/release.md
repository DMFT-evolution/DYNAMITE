# <img class="icon icon-lg icon-primary" src="/DMFE/assets/icons/tag.svg" alt="Release icon"/> Release Process

1) Update `CHANGELOG.md` (Keep a Changelog format)
2) Bump version macros or metadata in `include/version/*`
3) Ensure docs build (MkDocs + Doxygen) is green
4) Tag the release `vW.X.Y.Z`
5) Publish release notes with key changes and upgrade notes
